There are 5 types of the ordered list type attribute. That is 5 diffrent ways we can design an ordered list and the table below contains the 5 possible types:

Type Description
type="1" The list items will be numbered with numbers (default)
type="A" The list will be numbered with uppercase letteres
type="a" The list items will be numbered with lowercase letters
type="I"The list will be numbered with usppercas roman numbers
type="i"The list will be numbered with lowercase roman numbers
