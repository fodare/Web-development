HTML helps web developers group set of elements / items into a list. There are generally 2 types of list.
They are namely

  1. Ordered List
  2. Un-Ordered list
Another example is :
  • Ordered list
  • Un-Order list
As the name implies the, Un-ordered list would list the items in no specific form and would display the items in the list with a a bullet point at the beginning. The bullet point starts with a dark circle by default.
An ordered list would display the items in the list in a chronological orde, and usally marked in numbers by default.
Another concept of the HTML list is the description list. The description list is a type of HTML description list that is use to describe the item of the list, basically describing the item in the list, we can do this by using the dl tag and then the dt and dd tag inside the dl tag. The dl tag is used to start the list, the dt tag is used to give a name of the list and the dd tag is used to add description to the of the list. Example below:
Damilare
Author's name
Male
Autho's Sex
Estonia
Author's country
