JavaScript Arrays

They are used to store multiple values in a single variable. Example is:
var carNames = ["Mazda", "Bentley", "Honda", "Ford", "Toyota"];

Creating an Array

There are various methods to create an array.

  1. Method 1 : var arrayName = ["value1", "Value2", .....]
  2. Method 2 : var cars = new Array("value1", "Value2", ....)

Accessing the Elements of an Array

To access any value of an array. Use the index value of the array value / data. Like so

var myNames = ["Damilare", "Olatunbosun", "Daniel"]

To access Olatunbosun : myNames[1]

Changing an Array Element

You can use the index value to access the value to be changed and then pass in the new value
