HTML inks generally have 3 types of link colors depending on the action performed on the
Link. Below are the default color of HTML links:

  • Blue: Meaning the link has not been visited / clicked.
  • Purple: Meaning the link has been clicked / visited before.
  • Red: Meaning the link is active

It's also possible to design the way the links are displayed on the website. We can do this by using CSS.
