Another concept or attribute of a link is the Target. The target attribute tells the browser where to display the link.
The possible values for the Link tag are:

  • _self. Default. Opens the document in the same window/tab as it was clicked.
  • _blank. Opens the document/ the link in a new window or tab.
  • _parent Opens the document in the parent frame.
  • _top Opens the document in the full body of the window.

This are the examples of the target attribute for a link:
Self Target. This would display the link on the same tab.
blant Target. This would dispaly the link in a new tab / window.
Parent Target. This would display the Link in the parent frame.
Top Target. This would display the link on the full body of the page.
