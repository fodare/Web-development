The HTML form is used to create a form to collect the user input then processed in another page running in the server. This is done by the us of HTML form element which looks like < form > The user's input is then collected and procced on the server side. Below is a simple example of a form.

Simple HTML Form



The < form > element is used to create the form on the page and the input elements could range from text box, radio buttons, check boxes and so many more. The < input > element is one of the most important element of the form. The table below shows the possible types of the possible input types and what they do.

Type Description
< input type="text" >Dislays a single-line txt input field
< input type="radio" > Displays a radio button(for selecting one of many choices)
< input type="submit" >Displays a submit buttion (for submittig the form)
